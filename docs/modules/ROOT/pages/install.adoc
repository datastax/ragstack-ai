= Install {product}

Install {product} in a new Python virtual environment.

One way to create a virtual environment:

[source,bash]
----
python3 -m venv ragstack-venv
source ragstack-venv/bin/activate
----

Install {product}:

[tabs]
======

+
[source,bash]
----
pip3 install ragstack-ai

[tabs]

+

----

----

+
[source,console]
----

Version: 0.1.1
Summary: RAGStack
...

class langchain_community.document_loaders.blob_loaders.file_system.FileSystemBlobLoader(path: Union[str, Path], *, glob: str = '**/[!.]*', exclude: Sequence[str] = (), suffixes: Optional[Sequence[str]] = None, show_progress: bool = False)[source]¬∂

Load blobs in the local file system.

Example:

from langchain_community.document_loaders.blob_loaders import FileSystemBlobLoader
loader = FileSystemBlobLoader("/path/to/directory")
for blob in loader.yield_blobs():
    print(blob)

Initialize with a path to directory and how to glob over it.

Parameters

  • path ‚Äì Path to directory to load from or path to file to load. If a path to a file is provided, glob/exclude/suffixes are ignored.

  • glob ‚Äì Glob pattern relative to the specified path by default set to pick up all non-hidden files

  • exclude ‚Äì patterns to exclude from results, use glob syntax

  • suffixes ‚Äì Provide to keep only files with these suffixes Useful when wanting to keep files with different suffixes Suffixes must include the dot, e.g. ‚Äú.txt‚Äù

  • show_progress ‚Äì If true, will show a progress bar as the files are loaded. This forces an iteration through all matching files to count them prior to loading them.
